# 轮播图按钮改进计划

## 当前问题
当前轮播图指示器是简单的线条按钮，反馈感较弱：
- 只有颜色变化（白色 vs 40%透明度白色）
- 没有尺寸或形状变化
- 没有悬停效果
- 没有进度指示

## 改进方案

### 方案1: 进度条式指示器（推荐）
- 当前激活的指示器显示进度动画（从左到右填充）
- 视觉上明确告知用户当前幻灯片剩余时间
- 符合宝马官网的高级感

### 方案2: 圆形指示器 + 悬停放大
- 将线条改为圆点
- 当前激活的圆点放大并改变颜色
- 悬停时有轻微放大效果
- 点击时有按压反馈

### 方案3: 数字指示器
- 显示 "01 / 03" 格式的当前幻灯片编号
- 配合细线进度条
- 更工业、更专业的感觉

### 方案4: 箭头导航 + 指示器
- 添加左右箭头按钮
- 指示器改为小圆点
- 悬停箭头时显示下一张预览

## 建议实施步骤

1. **选择方案** - 与用户确认采用哪种方案
2. **修改 HeroCarousel.tsx**
   - 更新指示器样式
   - 添加悬停/点击交互效果
   - 可选：添加进度动画
3. **测试验证**
   - 检查视觉效果
   - 验证交互反馈

## 技术实现要点

- 使用 Tailwind CSS 实现过渡动画
- 使用 `group-hover` 实现悬停效果
- 使用 `transform scale` 实现放大效果
- 可选：使用 CSS animation 实现进度条填充
